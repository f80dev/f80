Production d'une API de clustering & représentation 3D pour le CNRS

L'api, développée en Python, repose sur le framework Flask. Elle utilise les derniers algorithmes de clustering en particulier HDBSCAN pour regrouper des échantillions archélogique par composition chimique proche.

Le résultat du clustering est ensuite projeté dans un espace à 3 dimensions par analyse en composante principale puis représenté en 3D dans n'importe quel navigateur WEB par la mise en oeuvre de la bibliotheque BabylonJS, habituellement utilisé pour le jeu vidéo.

Du point de vue infrastucture, l'api peut être déployé rapidement sur n'importe quel serveur x86 via Docker

Exemple d'utilisation :
https://ss.shifumix.com:5000/graph/lesmiserables.gml/fr?algo_comm=NOTREATMENT&dir=public¬ext=True&nometrics=True&add_property=False&autorotate=True&limit=5000&pca=1&processors=2
