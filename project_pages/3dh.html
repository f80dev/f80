3DH est un interface web responsive d'usage de l'API de clustering et représentation 3D développée pour le CNRS.

L'interface permet :
  • d'uploader les fichiers sur le serveur de l'API
  • d'afficher une preview des traitements effectués sur les données : clustering ou analyse de graph
  • de créer un lien partageable appelant directement l'API en fonction des paramètres des algorithmes préalablement déterminés
  • de gérer les FAQs
Cette interface repose sur le framework VueJS. Afin d'optimiser son coût de fonctionnement elle est hébergée totalement gratuitement sur les "github pages". Un site de présentation, également hébergé par github a été développé sur Bootstrap
