Une application web mobile de fidélisation, totalement personnalisable, permettant

  • la gestion d'un compte client (inscription, identification),
  • Le rappel de ses achats et les services associés
  • La gestion d'un compte de point de fidélité
  • La gestion de promotion, limitées dans le temps

Techniquement, SelfApp est une 'progressive web app', dévloppée :

  • pour le backend : sur un serveur Google via une base de données et des api en java,
  • pour le front-end : une progressive web app dévélopper en Angular 6
